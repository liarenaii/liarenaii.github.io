# [成长中的陪伴](https://liarenaii.github.io/)

纪念日前夕，她收到猎头的电话，有一个很好的工作机会，但在另一座城市。

晚餐时她提起，语气轻松，眼神却泄露了紧张。他夹菜的手顿了顿，然后继续把肉放进她碗里。

“你怎么想？”他问。

“我不知道。”她老实回答，“机会很好，但……”

但这里有他们刚装修好的房子，有共同建立的生活，有彼此。

那晚他们聊到凌晨，分析了所有可能：异地、一起搬迁、放弃机会。每个选项都有代价，没有完美答案。

最终她说：“我想试试。”

他说：“好。”

没有戏剧化的挽留或牺牲，只有两个成年人，在权衡之后做出的选择。她去了新城市，他留了下来。视频通话成了日常，他们分享各自城市的天气、午餐吃了什么、工作中好笑的事。

距离放大了某些东西——比如想念，也缩小了某些东西——比如不必要的计较。每次见面都像小小的节日，分别时在车站拥抱，她闻到他衣领上熟悉的洗衣液味道，突然就觉得一切值得。

一年后，她也为他争取到了同一城市的工作机会。

搬家那天，阳光很好。他们站在新公寓的阳台上，看楼下公园里孩子们奔跑。

“还记得我们第一次约会吗？”她突然问。

“记得，下雨天。”他握住她的手，“你头发上有阳光。”

时间改变了太多，但有些东西始终没变。爱情不是静止的雕塑，而是流动的河流，有时湍急，有时平缓，有时分岔，最终汇入更广阔的水域。

他们学会了在各自成长的同时并肩前行，明白了最好的陪伴不是形影不离，而是即使走向不同的方向，灵魂的坐标依然为彼此保留。

夜幕降临，城市灯火次第亮起。阳台上，两个身影靠在一起，安静地看着这个他们将要共同书写新故事的地方。

远处传来隐约的音乐声，像一首没有尽头的歌。
